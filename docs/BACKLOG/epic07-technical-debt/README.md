# Epic: Technical Debt

## Purpose

Tracks issues, improvements, and gaps discovered during implementation that do not fit
neatly into another epic or were deferred to avoid blocking progress.

## Status: Ongoing

## Dependencies

None — stories are added here as they are discovered.

## Rules

- Add a story here when you discover an issue during implementation that cannot be fixed
  immediately
- Never use this epic as a dumping ground for things that should be done now — if it
  blocks the current story, fix it now
- Review this epic at the start of each new epic to see if anything should be promoted
